URL Shortener Microservice

A simple API to shorten long URLs and redirect to them

Overview

This microservice allows users to shorten long URLs into a simple numeric code. The shortened URL will then redirect the user to the original long URL, making it easier to share and manage.

Create a Short URL

Enter a valid URL in the form below, and the service will generate a short URL for you:

URL Shortener Form

Example: POST [this_project_url]/api/shorturl with https://www.google.com

Redirect with Short URL

After creating a short URL, use it to get redirected back to the original URL. Here's an example:

[this_project_url]/api/shorturl/3

Will Redirect To:

https://github.com

Important Note:

This microservice is designed for demonstration purposes. It shows how URL shortening and redirection work through a simple API setup.
